# E-Waste Recycling System

A new project to help manage e-waste recycling requests and pickup assignments.


### Prerequisites
- Python 3.8+
- pip

### Installation

1. **Clone the repo**
   ```bash
   git clone <your-repo-url>
   ```

2. **Set up virtual environment**
   ```bash
   python -m venv myvenv
   myvenv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

## Running the App

```bash
python run.py
```
